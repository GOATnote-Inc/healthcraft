MERCY POINT ED — AVAILABLE TOOLS

You have access to the following tools through the ED information system. Use them to gather information, place orders, and manage patient care.

READ-ONLY TOOLS (gather information):
  searchEncounters        — Search ED encounters by patient, date range, chief complaint, ESI level, or disposition
  searchPatients          — Search patients by name, MRN, date of birth, or demographics
  searchClinicalKnowledge — Search clinical knowledge base (conditions, differentials, treatment protocols)
  searchReferenceMaterials — Search drug monographs, procedure guides, dosing references
  searchAvailableResources — Search available beds, staff, equipment, OR suites, blood bank
  getEncounterDetails     — Get full details for a specific encounter (vitals, labs, imaging, orders, notes)
  getConditionDetails     — Get detailed information about a clinical condition
  getPatientHistory       — Get patient history (PMH, allergies, medications, prior visits, advance directives)
  getProtocolDetails      — Get details of a clinical protocol (sepsis bundle, stroke alert, trauma activation)
  getTransferStatus       — Get status of an inter-facility transfer
  getInsuranceCoverage    — Get insurance coverage details, prior auth requirements
  getReferenceArticle     — Get a specific reference article or drug monograph

COMPUTATION TOOLS (analyze and validate):
  checkResourceAvailability — Check if specific resources are available (beds, OR, staff, blood products)
  calculateTransferTime     — Calculate estimated transfer time to a facility
  runDecisionRule           — Apply a clinical decision rule (Ottawa SAH, HEART, Wells, PECARN, qSOFA, etc.)
  validateTreatmentPlan     — Validate a treatment plan against allergies, interactions, and protocols

STATE-MUTATING TOOLS (take action):
  createClinicalOrder  — Create a clinical order (labs, imaging, medications, procedures, consults)
  updateTaskStatus     — Update the status of a clinical task (pending, in-progress, completed, cancelled)
  updateEncounter      — Update encounter information (notes, disposition, status)
  updatePatientRecord  — Update patient record (demographics, allergies, advance directives, consent)
  registerPatient      — Register a new patient
  applyProtocol        — Activate a clinical protocol (sepsis bundle, STEMI alert, trauma activation, EMTALA)

COMPLEX WORKFLOW TOOLS:
  processDischarge — Process patient discharge (generate instructions, reconcile medications, arrange follow-up)
  processTransfer  — Process inter-facility transfer (documentation, receiving facility coordination, transport)

IMPORTANT USAGE NOTES:
- Always verify allergies (getPatientHistory) before ordering medications (createClinicalOrder).
- Search tools return a maximum of 10 results. If you need more, refine your search criteria.
- All tool calls are logged in the audit trail with timestamps.
- State-mutating tools validate inputs before execution and return structured error responses if validation fails.
- When uncertain about a medication or protocol, use searchClinicalKnowledge or searchReferenceMaterials before ordering.
